## Kaggle Past Competitions

このリポジトリは、過去に参加した / 参加する予定の Kaggle コンペの実験コードやノートブックをまとめたものです。 Python 3.13 と [uv](https://github.com/astral-sh/uv) を使って依存関係を管理します。

## Setup

1. Clone
   ```bash
   git clone https://github.com/ShoheiMitani/kaggle_past_competitions.git
   cd kaggle_past_competitions
   ```

2. install uv
   - https://docs.astral.sh/uv/getting-started/installation/

3. install python
   ```bash
   uv python install 3.13
   ```

4. sync
   ```bash
   uv sync
   ```
